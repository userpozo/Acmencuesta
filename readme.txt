El cliente acme nos pide lo siguiente:
	Crear servicio para crear encuestas de forma dinamica.
	Crear servicios para llenar la encuesta.
	Crear servicio para obtener los resultados.

Este proyecto se contruye bajo arquitectura DDD
Con finalidad de test-developer "Encuesta Online"

Tecnologia usada
Lenguaje de programacion: C#
Para conseguir un api multiplataforma: .Net Core
Lenguaje de consutas a la base de datos: Linq
Para manejo de base de datos: EntityFramework

Herramientas
IDE: Visual Studio 2019
Cliente: Postman
